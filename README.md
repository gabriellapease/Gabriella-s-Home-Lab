# Gabriella-s-Home-Lab

A personal cybersecurity homelab documenting hands-on projects built to develop real-world skills in networking, Linux system administration, privacy, and security tooling.
> *"The best way to learn cybersecurity is to build it yourself."*

---

## 👤 About

I am a cybersecurity student actively building practical skills outside the classroom through hands-on homelab projects. This repository serves as a portfolio of everything I have built, configured, and troubleshot — including the mistakes and how I fixed them.

**Current Focus Areas:**
- Network security and DNS filtering
- Linux system administration
- Privacy hardening
- Homelab infrastructure
